## Features

- Choice-based narrative progression

## Tech Stack
- Frontend: React + TypeScript
- Backend: Node.js/Express
- Database: MongoDB
- AI: OpenAI API

## Setup
1. Clone repository
2. Install dependencies: `npm install`
3. Set up environment variables
4. Run development server: `npm run dev`
